# Agentic Data Analyst App

This application uses **CrewAI** and **Google Gemini** to analyze your data and provide strategic insights.

## Setup

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the App**:
    ```bash
    streamlit run app.py
    ```

3.  **API Key**:
    - You will need a Google Gemini API Key.
    - Enter it in the sidebar when the app launches.

## Features

-   **Data Preview**: See your data and basic statistics immediately.
-   **AI Crew**:
    -   **Data Analyst Agent**: Identifies patterns.
    -   **Business Strategist Agent**: Recommends actions.
-   **Interactive**: Upload any CSV or Excel file.
